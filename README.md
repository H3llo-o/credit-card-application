# Untitled Credit Card Application Form
*A full-stack web application for banks to simulate online applications and various transactions*

## About
- This is a web application built in partial completion for **Information Management**
- The focus is to apply a simple database
- It will demonstrate basic **CRUD functionalities** for a relational database

## Scope
- Covers basic CRUD functionalities
- The current build would support credit card applications for the mean time
- Applications can vary per application but an applicant can only apply for a single card at a time
- Applicants may update their credentials or remove an old record

## Tech Stack
*Structured as a full-stack application, this would offer a secure data pipeline and a manageable ui to satisfy potential clients*

### Frontend
- ***Framework:*** Next.js
- ***Libraries:*** React
- ***Styles:*** Tailwind CSS

### Backend
- ***Environment:*** Node.js
- ***Database:*** MySQL
- ***Connections and Languages:*** TypeScript, JavaScript, SQL
